# react 操作命令
- 创建vite工程
```npm
npm init vite@latest app-name --template react-ts
```
```
cd app_name
yarn install
yarn dev
```
显示如下
```vite
  vite v2.9.6 dev server running at:

  > Local: http://localhost:3000/
  > Network: use `--host` to expose

  ready in 317ms.
```
